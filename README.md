
Trabalho de Conclusão de Curso

Utilizarei este repositório para salvar as modificações do sistema que estou desenvolvendo para o TCC, como também para fazer o controle de versões. 


Versões

PHP 8.2.12
XAMPP v3.3.30
HTML5
PHPmyADMIN 5.2.1
Libmysql-Mysqlnd 8.2.12

Organização:

O sistema está organizado no modelo MVC e contém as seguintes pastas:

Conexao - Esta pasta contém a conexão com o banco de dados
Controller - Recebe as requisições do usuário e encaminham a resposta para as Views.
Model - Contém as entidades do sistema, cada classe representa uma tabela do banco.
View - Contém as telas do sistema (interface com usuário).
